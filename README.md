## 下载

### Docker

```
docker pull apachecn0/impatient-js-zh
docker run -tid -p <port>:80 apachecn0/impatient-js-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g impatient-js-zh
impatient-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```
